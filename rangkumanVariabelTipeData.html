Variabel & Tipe Data JavaScript Hana

Pengertian Variabel

Variabel adalah sebuah nama yang mewakili sebuah nilai. Variabel dapat menyimpan berbagai jenis data seperti string, number, object, array, dan sebagainya.

Cara Membuat Variabel


var title = "Belajar Pemrograman Javascript";
    

let price = 15000;
stock = 12;

Mengisi & Menampilkan Variabel


var age = 18;
age = 21;
console.log(age);

Menghapus Variabel


bookTitle = "Belajar Pemrograman Javascript";
delete bookTitle;

Mengenal Tipe Data

  • String (Teks)
  • Number / Integer
  • Float (Pecahan)
  • Boolean
  • Object
